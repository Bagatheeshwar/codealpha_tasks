## 🎯 Interactive Quiz Features

- **10 questions** covering all presentation topics
- Multiple choice with A/B/C/D options
- Instant feedback with detailed explanations after each answer
- Visual progress bar
- Score summary with grade: Expert / Proficient / Developing / Needs Work
- Answer review at end showing correct answers
- Retake functionality
- Pure HTML/CSS/JS — no server required, works offline

---

## 🔑 Key Phishing Topics Covered

- **Definition & Impact** — statistics, financial cost, prevalence
- **Attack Types** — email, smishing, spear phishing, whaling, vishing
- **Social Engineering** — urgency, fear, authority, impersonation
- **URL & Domain Analysis** — spotting spoofed domains, hovering links
- **Email Analysis** — sender verification, suspicious attachments
- **Best Practices** — MFA, password managers, reporting culture
- **Incident Response** — immediate steps if compromised
